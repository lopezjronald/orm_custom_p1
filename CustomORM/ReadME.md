# Project 1: Custom ORM

## Technologies Used:
* Maven
* Java 8
* JUnit
* IntelliJ
* GitHub

## Overview
### Custom ORM is able to do the following methods:
* Create and remove a table
* Create and remove a column from a table
* Create and remove a user
